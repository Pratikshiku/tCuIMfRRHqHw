### 三、功能描述
系统分为（身份）：普通用户(学生)、教师、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 首页
4. 论坛
5. 课程信息
6. 公告信息
7. 院校展示
8. 公共资料
9. 个人中心
10. 学生成绩
11. 活动报名
12. 学生请假
13. 选课信息
14. 学分增加记录
15. 交流申请


**教师功能：**
1. 登录
2. 注册
3. 修改密码
4. 个人信息
5. 课程信息
6. 活动报名管理
7. 学生成绩管理
8. 学生请假管理
9. 选课信息管理
10. 交流申请管理
11. 学分增加记录
12. 院校展示管理
13. 公共资料管理
14. 论坛管理



**管理员功能：**
1. 登录
2. 个人中心（修改密码、个人信息）
3. 教师管理
4. 学生管理
5. 课程信息管理
6. 活动报名管理
7. 学生成绩管理
8. 学生请假管理
9. 选课信息管理
